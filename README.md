# 学生信息管理系统(C++)

*基于VC6.0*

## 主要功能

- 读取文件
- 保存文件
- 查看数据结构
- 打印学生信息
- 学生记录的增删改查和排序

## 用到的知识

- 链表模版
- 数组
- 输出格式化
- 文件输入输出
- 菜单界面美化
